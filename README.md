# Word-Scramble-Game
The game randomly selects a word from a predefined list and scrambles its letters. The scrambled word is displayed to the player, who must then guess the original word. The player has a limited number of attempts to guess correctly. After each incorrect guess, the program informs the player of the remaining attempts. If the player guesses the word correctly, they win the round; otherwise, the correct word is revealed when attempts run out.
